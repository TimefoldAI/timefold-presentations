False positives

  • Hard score: Is the solution feasible?
  • Soft score: Is the solution accurate?

Test for false positives

  • Given:
    • English @ Monday 09:00-10:00 in room 1
    • Spanish @ Monday 09:00-10:00 in room 1
  • Expected: NOT OK
javavar timeslot1 = new Timeslot("Monday 9:00-10:00", ...);
var room1 = new Room("Room 1", ...);

verifier.verifyThat(MyConstraints::roomConflict)
  .given(
    new Lesson("English", ..., timeslot1, room1),
    new Lesson("Spanish", ..., timeslot1, room1))
  .penalizes(1);

Test for false negatives

  • Given:
    • English @ Monday 09:00-10:00 in room 1
    • Math @ Monday 09:00-10:00 in room 2
  • Expected: OK
javavar timeslot1 = new Timeslot("Monday 9:00-10:00", ...);
var room1 = new Room("Room 1", ...);
var room2 = new Room("Room 2", ...);

verifier.verifyThat(MyConstraints::roomConflict)
  .given(
    new Lesson("English", ..., timeslot1, room1),
    new Lesson("Math", ..., timeslot1, room2))
  .penalizes(0);
